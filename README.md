# C# Intermediate Oppgave 3: Drone Race

[Refleksjoner](refleksjon.md)

## Kjøring

I DroneRaceApp så er det et menuprogram som kan startes med `dotnet run`.
Herfra kan alt kjøres.

Kjøring av prosjektet DroneRace med `dotnet run` kjører del A til C ellerhverandre.

ControlTowerAPI kan startes med `dotnet run` i katalogen.

## Testing

I DroneRaceApp er det en TestTower.http fil som kan brukes med VSCode extensions som
"REST Client" for å teste serveren etter at den er startet.
